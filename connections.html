Como conectar
1
Configure o servidor bot
Em Configurações → URL do Bot, insira o endereço onde o servidor Baileys está rodando.
2
Clique em "Conectar Bot"
O servidor irá gerar um QR Code que aparecerá nesta tela.
3
Escaneie com o WhatsApp
Abra o WhatsApp no celular → Dispositivos conectados → Conectar um dispositivo → Aponte para o QR.
4
Bot pronto!
Após escanear, o status muda para Conectado e os disparos começam automaticamente.
